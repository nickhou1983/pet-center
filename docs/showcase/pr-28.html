🔍 M6 混合搜索模块

用照片 + 文字描述自动匹配，从宠物库智能召回最相似的记录

PR #28 · 功能展示

什么是混合搜索？为什么需要它？

传统搜索靠关键词精确匹配，难以处理模糊查询。混合搜索结合图像相似度和文本语义理解， 让用户用「上传一张照片」或「描述一下这只宠物」就能秒速找到最匹配的记录。无论是寻找走失宠物、查找领养对象， 还是浏览已备案宠物，都能用最自然的方式搜索。

核心能力亮点

📷图像相似度

上传宠物照片，系统自动用 CLIP 向量引擎对比，从宠物库找出视觉最相似的记录。

📝文本语义理解

输入「橘白色的小型田园猫」这样的自然语言描述，AI 理解语义后精准匹配。

🎯融合加权搜索

同时用图片和文字？系统自动计算加权融合分数，取两者优势得到最佳结果。

🏷️属性精筛

结合物种、大小、性别、地区等属性过滤，搜索结果更聚焦，分页展示便于浏览。

三种自适应搜索模式

根据用户输入自动判定，无需手动切换：

图搜图
仅上传照片 → 系统用 CLIP 提取图像特征，对宠物库中每只宠物的图像向量做余弦相似度比较， 返回视觉最相近的 TOP-N 宠物记录。
文搜图
仅输入文字 → 文本向量化后，对宠物库的图像向量做相似度对比（CLIP 的跨模态能力）。 例如「三花母猫」描述会匹配到三色花纹的宠物照片。
融合搜索
图片 + 文字都有 → 分别获得两路分数，按用户调节的权重（默认各 50%）加权融合。 系统自动在数据库内完成计算与排序，保证分页结果准确。

技术实现亮点

深度集成 PostgreSQL pgvector + CLIP 跨模态向量引擎：

  • 矢量化在 SQL 内 — 融合分数直接在数据库计算（`score = wImage·imageScore + wText·textScore`）， 保证 `ORDER BY score DESC` 与 `LIMIT/OFFSET` 分页的准确性
  • 安全的参数化查询 — 所有 SQL 均用 `Prisma.sql` 参数化，文本筛选自动转义通配符， 完全杜绝 SQL 注入
  • 零新依赖 — 不新增第三方包，复用现有 M2 数据层 / M3 CLIP 向量能力
  • 前后端统一校验** — 共享 zod 模式（`lib/search-schema.ts`），确保类型安全与一致性
  • 跨模态魔法 — CLIP 同时理解图像和文本，用同一列向量对比，实现图文互搜

API 契约详解

📤 请求示例

POST /api/search

curl -X POST http://localhost:3000/api/search \ -H "Content-Type: application/json" \ -d '{ "description": "橘白色的小型田园猫", "category": "REGISTERED", "species": "CAT", "size": "SMALL", "page": 1, "pageSize": 20 }'
📥 响应示例 (200 OK)
{ "mode": "text", "weights": { "image": 0, "text": 1 }, "page": 1, "pageSize": 20, "total": 5, "results": [ { "id": "pet-001", "name": "橘橘", "species": "CAT", "size": "SMALL", "score": 0.92, "imageScore": null, "textScore": 0.92 } ] }
支持的字段与错误码
  • 查询输入：`photo`（`/uploads/*` 路径）、`description`（文本）—— 至少填一项
  • 权重调节：`wImage`、`wText`（默认 0.5/0.5，两路都有时自动归一化）
  • 属性筛选：`category`、`species`、`size`、`gender`（枚举精确匹配）； `breed`、`color`、`region`（大小写不敏感子串）
  • 分页：`page`（默认 1）、`pageSize`（默认 20，上限 50）
  • 错误响应：`400 VALIDATION_ERROR`（校验失败）、`400 INVALID_PHOTO`（照片路径不合法）、 `502 EMBEDDING_FAILED`（向量生成失败）、`500 DB_ERROR`（查询失败）

实际应用场景

  • 🐱 寻找走失宠物 — 拍一张似曾相识的野猫/野狗照片，或描述其特征，立即搜索是否有走失记录
  • 🏠 领养前查重 — 上传想领养的宠物照片，检查是否已在其他平台/渠道有记录
  • 📋 备案查询 — 输入「黑白边牧」「大型黄金猎犬」等描述词，快速定位已备案的宠物信息
  • 🎓 数据分析 — 基于搜索热度和匹配分数，了解用户对宠物的兴趣分布和特征偏好
